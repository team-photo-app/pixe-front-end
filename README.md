# An events pic app

PixE is an event pix app that allows users to create events and have guests capture and share pictures.

## Installation and Scripts

Use npm to install dependencies.

```
npm install

"scripts": {
    "start": "expo start",
    "android": "expo start --android",
    "ios": "expo start --ios",
    "web": "expo start --web",
    "eject": "expo eject"
  },
```
